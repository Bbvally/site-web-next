## Implementation Guide

### Quick Creation (30 min per page)

1. **Copy template** from completed page (e.g., restaurants)
2. **Replace colors** with industry-specific color scheme
3. **Update H1 & descriptions** for industry
4. **Customize pain points** (5 items)
5. **Customize solutions** (5 items)
6. **Update benefits** (6 boxes with icons)
7. **Adjust pricing** based on industry
8. **Write FAQ** (3-5 industry-specific questions)
9. **Update CTA** messaging
10. **Test & deploy**

### Using Template Code

```typescript
// Change this:
const metadata: Metadata = {
  title: "Google Reviews for Restaurants Nigeria",
  description: "..."
};

// To this:
const metadata: Metadata = {
  title: "Google Reviews for [INDUSTRY] Nigeria",
  description: "Get authentic Google reviews for your [INDUSTRY] in Nigeria..."
};
```

### Key Customization Points

1. **H1 Headline** - Change to industry
2. **Meta title/description** - Include industry keyword
3. **Color scheme** - Change gradient and button colors
4. **Pain points** - List 5 industry-specific challenges
5. **Benefits** - Update 6 benefit boxes with industry benefits
6. **Pricing** - Adjust based on industry (healthcare higher, retail lower)
7. **CTA** - Industry-specific call-to-action

## Technical Notes

### Metadata
Each page should include:
```typescript
export const metadata: Metadata = {
  title: "Google Reviews for [INDUSTRY] Nigeria",
  description: "Industry-specific value prop",
  alternates: {
    canonical: "https://buyreviewsinnigeria.com/industries/[slug]",
  },
};
```

### Schema Markup
Each page includes:
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "[INDUSTRY] Google Reviews Service",
  "description": "...",
  "provider": {
    "@type": "Organization",
    "name": "BuyReviewsInNigeria"
  }
}
```

### Color Implementation
Use Tailwind CSS classes:
```jsx
bg-[color]-50    // Light background
text-[color]-600 // Primary button
border-[color]-600 // Accents
hover:bg-[color]-700 // Hover state
```
