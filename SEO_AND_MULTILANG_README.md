## 🆘 Troubleshooting

### Issue: "Module not found: next-intl"
**Solution:** Run `npm install next-intl`

### Issue: Pages not appearing in [locale] structure
**Solution:** Ensure directory is named `[locale]` with square brackets

### Issue: Translations not loading
**Solution:** 
- Check file names: `en.json`, `fr.json`, `yo.json`, `ha.json`
- Verify path: `src/messages/`
- Validate JSON syntax

### Issue: hreflang tags missing
**Solution:** Add to page metadata:
```typescript
alternates: {
  languages: {
    "en-NG": "https://...",
    "fr-NG": "https://...",
  }
}
```

**More troubleshooting:** See MULTI_LANGUAGE_IMPLEMENTATION.md
